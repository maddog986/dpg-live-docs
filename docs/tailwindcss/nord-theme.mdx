# TailwindCSS Nord Theme

[Nord Theme](https://www.nordtheme.com/) impsired color combinations for [TailwindCSS](https://tailwindcss.com).

## Config

Basic config of the additional colors.

```js
module.exports = {
  theme: {
    extend: {
      colors: {
        // https://www.nordtheme.com/
        night: {
          '400': '#2E3440',
          '300': '#3B4252',
          '200': '#434C5E',
          '100': '#4C566A'
        },

        snow: {
          '100': '#ECEFF4',
          '200': '#E5E9F0',
          '300': '#D8DEE9'
        },

        frost: {
          '100': '#8FBCBB',
          '200': '#88C0D0',
          '300': '#81A1C1',
          '400': '#5E81AC'
        },

        aurora: {
          '100': '#BF616A',
          '200': '#D08770',
          '300': '#EBCB8B',
          '400': '#A3BE8C',
          '500': '#B48EAD'
        },
      }
    }
  }
}
```
