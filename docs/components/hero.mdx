# Hero

Create pro-quality landing page with a beautiful hero section.

## Code

```jsx live noInline
const Hero = ({ title, text, buttons = [] }) => {
    return (
        <div className="flex justify-center py-24 bg-gray-100">
            <div className="max-w-2xl p-12 text-center">
                <div className="text-3xl font-bold md:text-3xl">{title}</div>
                <div className="mt-4 text-xl font-normal">{text}</div>
                {buttons.length > 0 && <div className="relative flex justify-center h-12 mt-6">
                    {buttons.map(({ text, href }, idx) => (
                        <a key={idx} href={href} className="absolute flex px-4 py-2 text-lg font-medium text-white bg-green-600 rounded shadow-md cursor-pointer hover:underline tr-mt svelte-jqwywd">
                            {text}
                        </a>
                    ))}
                </div>}
            </div>
        </div>
    )
}

render(
  <Hero
    title={"Want to discover more?"}
    text={"Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor."}
    buttons={[{ text: "Join Us Now", href: "#" }]}
  />
)
```
